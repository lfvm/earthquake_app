
# Earthquake Data App

Android app that shows the user recent earthquakes around 
the world.

## Features

- Light/dark mode toggle
- Live previews
- Fullscreen mode
- Cross platform


## Tech Stack

**Client:** React, Redux, TailwindCSS

**Server:** Node, Express
